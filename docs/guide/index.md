# Getting Started

Welcome to the brickene documentation! This guide will help you get started with brickene and show you how to use its features effectively.

## Overview

brickene – A flow-graph based visual builder for block-assembled organic molecules.

This package provides a set of tools to help you build robust Python applications with ease. Whether you're building a CLI tool, a web service, or a data processing pipeline, brickene has you covered.

## Prerequisites

Before you begin, make sure you have:

- Python 3.9 or higher installed
- pip or Poetry for package management
- Basic knowledge of Python programming

## Installation

You can install brickene using pip:

```bash
pip install brickene
```

Or if you prefer using Poetry:

```bash
poetry add brickene
```

## Basic Usage

Here's a simple example to get you started:

```python
from brickene import example

# Simple greeting
result = example.hello("World")
print(result)  # Output: Hello World!
```



## CLI Usage

brickene also provides a command-line interface:

```bash
brickene --help
```

This will display all available commands and options.

Example usage:

```bash
brickene --name "Your Name"
```
